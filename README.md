# Frontend-Assignment

This project utilizes the SpaceX API to fetch and display capsule details.

## Features:
- > Login-Signup
   - > (Authentic user can only visit homepage)
- > HomePage
- > Filter By :
    - > Type
    - > Status
    - > Date

## Tech-Stack:
- > Front-end
    - > React JS, CSS, Context-API

- > Back-end
    - > Node JS, Express JS, mongoDB (Authentication)
